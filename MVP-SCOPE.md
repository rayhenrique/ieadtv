# Escopo do MVP (Produto Mínimo Viável)

## IN Scope (O que faremos agora)
### Prioridade Máxima (Must Have)
- Portal Público responsivo (Mobile-first na visualização, Desktop-first na gestão).
- Supabase Auth configurado para a equipe de mídia.
- CRUD de Notícias, Categorias e Banners.
- Embed do YouTube na Home.
- CRUD de Congregações e página de listagem pública.

### Prioridade Média (Should Have)
- Integração via API do YouTube para puxar o vídeo mais recente automaticamente (sem a equipe precisar colar o link manualmente toda vez).
- CRUD de Eventos (Agenda).

## OUT Scope (O que NÃO faremos no MVP)
- Login/Área de Membros para o público geral.
- Sistema de doações online/Gateway de pagamento (apenas dados de PIX em texto, se necessário).
- Perfis de permissão complexos (ex: Dirigente X só edita a Congregação X). Todos do painel terão acesso total (Superadmin).
- Hospedagem de vídeos no Supabase.

## Hipóteses a Validar
- A interface limpa e rápida aumentará o tempo de retenção na leitura dos artigos.
- A equipe de mídia conseguirá gerenciar os 5 módulos sem gargalos operacionais.
